## 🔎 What you will see

sherlock shows a list of sites and reports which ones match the username. For each site, it may show one of these results:

- Found
- Not found
- Error
- Rate limited

A found result means the username exists on that site. A not found result means it did not find that name there. If you see an error, try again later or check your internet connection.
